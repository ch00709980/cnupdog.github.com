标准输入输出
--------------
> 在unix环境下执行一个shell命令行通常会自动打开三个标准文件：
>> 标准输入文件```stdin```:***对应终端的键盘终端***   
>> 标准输出文件```stdout```:***对应终端的屏幕***   
>> 标准错误输出文件```stderr```:***对应终端的屏幕***      

标准流是当一个程序运行后，在程序和它的环境（典型终端），事先连接的输入和输出管道。当然管道的特点是单向的，管道把一个进程的输出传给另一个进程作为输入。
